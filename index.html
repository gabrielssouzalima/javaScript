JAVA SCRIPT - HISTÓRIA

1. Java Script e Java são a mesma coisa?

Não. Java é uma linguagem de programação OOP, ao passo que Java Script é uma linguagem de scripts OOP. Java cria aplicações executadas em uma máquina virtual ou em um browser, ao passo que o código JavaScript é executado apenas em um browser.

2. Quando surgiu o Java Script?

O JavaScript logo chegou ao Netscape Navigator 2.0 em setembro de 1995.

3. Quem foi o responsável pela criação do Java Script?

O JavaScript foi concebido como uma linguagem do lado do servidor por Brendan Eich (então empregado pela Netscape Corporation).

4. Qual o caminho trilhado pelo responsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?

Após desenvolver o JavaScript na Netscape, Eich ajudou a fundar a Mozilla Foundation - criadora do navegador web Mozilla Firefox - e então, fundou a Brave Web Browser. Atualmente a empresa Netscape foi comprada pela AOL em meados de 1999.
